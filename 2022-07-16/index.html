misc: asdf

anyenv でシェルの起動が遅いので、asdf に変更した。

  • プラグイン一覧
    • asdf plugin list all
  • インストール済みのプラグイン一覧
    • asdf plugin list
  • プラグインのインストール
    • asdf plugin add nodejs
  • プラグインのバージョン一覧
    • asdf list all nodejs
  • プラグインのインストール済みバージョン一覧
    • asdf list nodejs
  • バージョンをインストール
    • asdf install nodejs 16.7.0
  • ローカルのバージョンを固定
    • asdf local nodejs 16.7.0
  • 現在のバージョンを表示
    • asdf current
